# Site "Juntos desde 12/07/2025"

Pequeno site estático que mostra quanto tempo (dias/horas/minutos/segundos) um casal está junto desde 12/07/2025 e permite escolher 4 fotos que ficam salvas no navegador.

Como usar:

- Abra o arquivo `index.html` no navegador (duplo-clique ou arraste para uma aba do navegador).
- Clique nos quadrados "+" para escolher cada foto; elas serão salvas no armazenamento local do navegador.
- Use o botão "Limpar fotos" para remover as imagens salvas.

Observações:

- As fotos são salvas no armazenamento local do navegador (localStorage). Elas ficam apenas neste computador e navegador.
- Se preferir colocar imagens na pasta images, posso alterar o código para usar arquivos locais estáticos em vez de upload.
